Medidor de temperatura

El sensor de temperatura mide continuamente la temperatura ambiente, tiene un microcontrolador recibe las lecturas del sensor y las compara con el umbral establecido (en este caso, 32 grados Celsius), si la temperatura supera el umbral, el microcontrolador activa el relé, permitiendo que la corriente llegue al ventilador, que se enciende, y al bajar la temperatura el microcontrolador desactiva el relé, apagando el ventilador.
